==1.移除用完的標籤紙==
==2.從貼標機旁邊的紙箱拿取標籤紙==

==3.這橫桿容易有殘膠,請拿酒精跟濕紙巾擦拭==
![[../attachments/Pasted image 20251013151257.png]]

==4.將箭頭處這卡榫往下扳==
![[Pasted image 20250912101600.png]]

==5.向下扳後卡榫會打開
(如標籤紙裝回後,未把卡榫裝回,標籤機螢幕會出現E01 HeadOpen的錯誤)==
![[Pasted image 20250912101720.png]]

轉動黑色旋鈕,移開這Y型支架,接著將全新的標籤紙裝上
![[Pasted image 20251107090145.png]]

==6.裝上標籤紙的方向如下圖,全新標籤紙裝上後,將標籤紙撕八張起來,留下方的底紙即可,等等再安裝時會比較好安裝
將標籤底紙裝上後,拉長並放置在紅圈處的橫桿上,此圖為剛換上新的標籤紙畫面==

![[../attachments/Pasted image 20251013145529.png]]

7.標籤紙擺放位置請依照箭頭指示
![[Pasted image 20251104124254.png]]
8.標籤紙走向請參照紅色路線
![[Pasted image 20251104124419.png]]

9.依照圖片所示
![[Pasted image 20251104124909.png]]
![[Pasted image 20251104124600.png]]


10.在擺放標籤紙時,需將紙擺放在定位塊上(紅圈處)
![[Pasted image 20251104130041.png]]

11.標籤紙需遮到此電眼,感測紙張用,標籤紙需平整
![[Pasted image 20251104130316.png]]

12.紙張上來後須壓在黑色滾輪上面
注意紅圈處壓紙的滾輪,需緊實壓在標籤紙上,如無,需調整

![[Pasted image 20251104130959.png]]
下圖為正常情況
![[Pasted image 20251104131637.png]]

13.如需調整滾輪角度,使用六角板手,放鬆紅圈處六角螺絲,就可以調整滾輪角度了
![[Pasted image 20251104131753.png]]

14.將標籤紙底紙以逆時針裝上,需裝在銀色鐵桿上(長方形紅框處),裝上後,
逆時針旋轉幾圈,讓標籤紙底紙能夠固定,如第二張圖
![[Pasted image 20251104133715.png]]
![[Pasted image 20251104163111.png]]

15.旋轉回收轉盤,使下方有標籤貼紙的位置,轉動到如下的位置
(兩張貼紙的間隙,大約落在左邊螺絲再偏右一點點
![[Pasted image 20251107091248.png]]

16.裝上Y型支架
![[Pasted image 20251107090145.png]]

17.關掉電源再重開,讓包
